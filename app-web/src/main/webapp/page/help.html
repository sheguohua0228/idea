• 1、APP有时打开闪退或更新后闪退?

    首先确定APP是从正规引用市场下载，在手机设置里找到乐E下清理数据和缓存。确保网络通畅的情况下打开APP。

  • 2、使用优惠码时显示优惠码无效

    如果没使用过该优惠码的情况下显示无效，请致电客服。

  • 3、衣服下单后多久可以上门取件,多久可以送回?

    15分钟上门取件,72小时内送回。但个别需要特殊洗涤和处理的衣物可能会有延迟，超过72小时才能送回的衣服，我们工厂会提前电话跟您沟通。皮革类、鞋包清洗护理的时间要长些，需要5-7天左右。

  • 4、如果我的衣物在洗的过程损坏或丢失怎么办?

    若衣物在清洗过程中出现损坏、丢失等事故，请你立即拨打我或者400电话，告知订单信息，我会联系工厂为您进行查询。乐E洗衣会在7天内给到用户满意的答复：快速核实后根据消协洗赔付标准对用户进行先行赔付。

  • 5、如果对洗衣效果不满意是否可以重新洗?

    可以，请您直接拨打您社区的乐E小哥电话或我们的服务热线 4008 323 325，我会为您填写返洗订单，上门收取衣物为您进行返洗。建议您在收到衣服的48小时内，去水洗标及穿着前完成检查。

  • 6、积分怎么使用?

    新用户注册即送50积分，消费1元钱则可获得1积分，积分可以直接抵现，您可以用积分直接消费，打印照片、洗衣等。

  • 7、衣服是否分类清洗,是否与别人的衣物混合?

    衣服送到洗衣工厂后，会根据面料和颜色进行分拣，进行不同方式的处理（干洗、水洗、湿洗、手洗等）。每件衣服都经过了12道专业清洗工序，清洗后会进行专业的杀菌消毒过程，不同面料的衣服会用不同的方法进行烘干消毒处理（高温烘干、低温摇摆式烘干等）。无论哪种烘干方法，都可以保证比家庭洗衣更清洁，并避免了雾霾空气的二次污染。

  • 8、乐E下是如何提供洗衣服务的?

    取送环节：公司自建物流体系，乐E下衣物取送工作都交给您所在社区里有固定住所的“乐e小哥”。他们全部是乐E下公司长期聘用的员工，您一键下单后，咱乐e小哥踩着七彩祥云以最快的速度为您上门取送衣服。下一步，我们即将在社区陆续开通“乐e代收”、景区门票\电影票打印等业务，您的麻烦，我们解决！

  • 9、照片怎么打印?

    您在我们“乐E下”APP选取照片、选择尺寸、打印张数后，支付成功后系统自动分配“提取码”，您可以查看“乐E终端地图”，到离您最近的乐E智能终端社区选择“属于你的精彩”输入提取码，就可以拿到您的照片啦。

  • 10、付了款但是我的照片没有打印出来怎么办?

    请您立即拨打我们400的客服热线，告知您的订单号和乐E终端位置，我们将尽快为您解决。

  • 11、为什么说我们的照片是高品质的数码专业照片?

    我们采用的是比喷墨打印成本高出很多的热升华打印技术，使用富士或Kodak相纸，其专用色带含有特殊保护膜，在防水防紫外线及防指纹的表现上占有更大优势，其256级色阶使照片色彩鲜活、层次分明，能保持相片颜色鲜艳，五十年不退色。

  • 12、为什么我的照片打印出来不清楚?

    请尽量上传照片原片，如果是微信、QQ等网络下载的照片，由于其照片的像素和分辨率经过了压缩，一张3M的照片经过压缩后只有几十K，自然打印的照片就模糊了。

  • 13、为什么手机上清晰的照片打印出来是模糊的?

    因为现阶段手机屏幕平均在“1080P“的水准。以4×6寸照片为例，以分辨率为300dpi计算，原图至少需要200万象素，而且手机的屏幕亮度较高，如果将环境光计算在内图片由于过暗造成噪点过多，也会造成照片打印出来模糊。
